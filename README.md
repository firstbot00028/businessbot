# 🦅 Aira Pro Utility Bot (WhatsApp)
### Open Source Business Assistant by: Engineer Adam 💎

A logic-driven, high-performance WhatsApp automation bot designed to replace complex AI with fast, reliable utility functions. Perfect for small businesses and developers.

## ✨ Features
- **📊 Interactive Menu:** Triggered by `.menu` for easy navigation.
- **🚨 Urgent Alert:** Direct notification to admin for priority queries.
- **📅 Smart Booking:** Simulated booking flow for services.
- **📍 Location Sharing:** Instantly send office location and hours.
- **⚡ Zero-Lag Logic:** Runs on Node.js without the need for expensive AI APIs.

## 🛠️ Setup Instructions
1. `npm install whatsapp-web.js qrcode-terminal`
2. Replace `YOUR_NUMBER` in the code with your actual WhatsApp number.
3. Run `node app.js` and scan the QR code.
